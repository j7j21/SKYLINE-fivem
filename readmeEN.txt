NISSAN SKYLINE GT-R(BNR34)
Original Author: Turn 10 Studios(Forza Motorsport 4)
Converted to GTA5 by: y97y
- HQ exterior and HQ interior
- Tunable
- 2 bodykit "TopSecret" & "S-Tune"
- Working meters Brakebale glass & No tint lightglass
----------------------------------------------------------------
Installation:
MAKE SURE TO BACKUP YOUR FILES!!!
----------------------------------------------------------------
1) Use OpenIV extract 
X:\Grand Theft Auto V\update\update.rpf\common\data\dlclist.xml
then use notepad open it,add new line 
<Item>dlcpacks:\skyline\</Item>
Save it and use OpenIV replace it.
----------------------------------------------------------------
2) Use OpenIV extract 
X:\Grand Theft Auto V\update\update.rpf\common\data\extratitleupdatedata.meta
then use notepad open it,add new line
<Item type="SExtraTitleUpdateMount">
			<deviceName>dlc_SKYLINE:/</deviceName>
			<path>update:/dlc_patch/SKYLINE/</path>
		</Item>
Save it and use OpenIV replace it.
----------------------------------------------------------------
3) Use OpenIV
X:\Grand Theft Auto V\update\update.rpf\dlc_patch\
create a folder named skyline,then add content.xml in it.
----------------------------------------------------------------
4) Copy skyline folder to 
X:\Grand Theft Auto V\update\x64\dlcpacks
----------------------------------------------------------------
That's all Enjoy it! 
You can use Simple Trainer Spawn it by name SKYLINE
